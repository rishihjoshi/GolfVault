01 Order Management
Shopify wins ▾
Feature ● Shopify ● Custom Build Analysis
Order creation & status tracking
Customer-facing order history, status updates, fulfilment states
 ✓ Native Built by Bihayy Shopify's OMS is production-hardened across millions of stores. Bihayy's custom implementation will work but will take 3–4 weeks to build and test properly.
Order fulfilment & shipping labels
Integration with USPS, UPS, FedEx for St. Louis delivery
 ✓ Native Custom API Shopify Shipping connects USPS, UPS, FedEx with one click. Labels generated in the admin. Custom requires a separate EasyPost or ShipStation API integration — not in Bihayy's $4,870 scope.
Returns & exchanges workflow
Customer-initiated returns, return labels, refund processing
 ✓ Native Custom logic Shopify handles return authorisation, return label generation, and automatic refund to original payment method. Custom build requires building all of this from Stripe's refund API — time-consuming, error-prone, not in current SOW.
Inventory management & stock alerts
Stock levels, low-stock alerts, backorder handling
 ✓ Native Supabase table Custom build stores inventory in Supabase — functional but requires building decrement logic on purchase, low-stock notification triggers, and admin alerts. Risk: race conditions on simultaneous purchases of the last unit in stock.
Discount codes & promotions
Promo codes, percentage/fixed discounts, expiry dates
 ✓ Native Via Stripe Stripe Coupons handles basic discount codes. Shopify's discount engine handles stacking, first-order, cart minimum, and BOGO — significantly more flexible for promotions.
Tax calculation (US + Canada)
State/province sales tax, automated filing reports
 ✓ Shopify Tax Stripe Tax Both handle automated US/Canada tax calculation. Shopify Tax includes nexus reporting. Stripe Tax charges 0.5% per transaction. For GolfVault's volume, Stripe Tax is adequate — not a deciding factor.
Multi-line order management dashboard
Admin view: all orders, filter by status, bulk operations
 ✓ Native Built by Bihayy Shopify admin is fully featured on day one. Custom admin dashboard is in Bihayy's scope but only covers basic status views — bulk operations, CSV export, and advanced filtering are not specified in the SOW.
Order email notifications
Confirmation, shipped, delivered, refunded transactional emails
 ✓ Native ✓ Resend Both handle transactional order emails. Custom uses Resend (already in the stack). Shopify's templates are pre-built; custom requires designing and maintaining email templates separately.
Corporate / B2B bulk order flow
Custom pricing, PO numbers, net payment terms, branded quotes
 Shopify B2B (Plus only) ✓ Custom built Shopify B2B requires Shopify Plus at $2,300/month — completely out of scope for GolfVault's stage. The custom corporate enquiry form and quote workflow in the GolfVault architecture handles this natively without additional cost.
Subscription order management
Recurring billing, pause/cancel, upgrade/downgrade
 Shopify Subscriptions app ✓ Stripe Billing Shopify requires a third-party subscriptions app ($20–$50/month) for recurring billing. Custom build uses Stripe Billing natively — already in the stack, no extra cost, and handles coaching memberships and video subscriptions in one system.
02 Catalog Management
Custom wins ▾
Feature ● Shopify ● Custom Build Analysis
Product catalog (physical accessories)
SKUs, variants (size/colour), images, descriptions
 ✓ Native ✓ Payload CMS Both handle standard product catalogs well. Shopify is faster to set up. Payload CMS requires initial schema configuration but gives more flexibility in content structure.
Coaching session catalog
Session types, durations, coach, price, availability
 ✗ Not designed for this ✓ Native Shopify's product model is physical-goods-centric. Coaching sessions can be forced into Shopify as products, but availability, coach assignment, and Cal.com integration require workarounds that add significant complexity and break cleanly. Custom build handles coaching sessions as a first-class catalog entity.
Video lesson catalog
Course titles, descriptions, Mux video IDs, access tiers
 ✗ Requires app ✓ Payload + Mux Shopify has no native digital video product. Would require Sendowl or similar ($19+/month) and cannot integrate with Mux for streaming access control. Payload CMS stores video metadata with Mux asset IDs. Access gating is handled by subscription entitlements in Supabase — no third-party app needed.
Non-technical content editing
Marketing team edits product descriptions, coach bios, blog posts
 ✓ Shopify Admin ✓ Payload Admin Both provide non-technical editors with a clean admin UI. Shopify's is more polished and immediately familiar. Payload's admin is auto-generated from your schema — slightly more learning curve but equally capable once configured.
Cross-catalog bundling
"Buy gloves + book a lesson" bundles, accessory + coaching packages
 Bundles app ($) ✓ Native logic Shopify bundles require a third-party app and cannot cross system boundaries — a bundle of a physical product + a coaching session + a video subscription is architecturally impossible in Shopify without significant custom development. Custom build creates bundles as first-class entities across all three product types.
SEO-optimised product pages
Meta titles, descriptions, structured data, canonical URLs
 ✓ Native ✓ Next.js SSG Both produce SEO-capable product pages. Shopify's are fully automatic. Custom Next.js SSG pages require implementation but are already in the architecture — and offer more structural data control (JSON-LD schema, Open Graph).
Algolia search integration
Fast product search across accessories, sessions, and courses
 Shopify → Algolia connector ✓ Direct Algolia Shopify's Algolia connector syncs product catalog only — coaching sessions and video courses are invisible to search. Custom Algolia integration indexes all three catalog types in a unified index — one search box finds gloves, lessons, and video courses simultaneously.
Custom product attributes
Golf-specific: shaft flex, grip size, handicap range, customisation options
 Metafields (limited) ✓ Unrestricted Shopify Metafields handle custom attributes but have display limitations in themes. Custom Payload CMS schema supports any field structure — handicap range for coaching packages, shaft specs for accessories, access tier for video courses — all natively.
Corporate branded catalog
Customisable accessories with company logo, dedicated corporate product pages
 Requires custom app ✓ Custom workflow Shopify's standard catalog cannot handle a "submit logo, preview branded product, get a quote" workflow without custom Shopify app development — which costs more than the custom build itself at this scale. GolfVault's custom corporate enquiry flow handles this natively.
Catalog data ownership & portability
Export, migrate, or repurpose catalog data freely
 CSV export only ✓ Full ownership Shopify owns your store data. Migrating away from Shopify requires CSV exports and manual re-import — coaching sessions and video metadata are not exportable. Custom Supabase + Payload data is yours entirely — portable, queryable, and accessible via API at any time.

When to choose each
The honest answer isn't binary. It's about which parts of GolfVault are a standard ecommerce problem and which parts are unique.
🛒 Choose Shopify when...
  • GolfVault is accessories-only and coaching is a secondary add-on with no digital delivery
  • You need to start taking orders in weeks, not months
  • Budget for upfront dev work is tight and ongoing platform fees are acceptable
  • The team has no technical capacity to manage a custom deployment post-launch
  • Returns, fulfilment, and inventory automation are higher priority than cross-sell
  • You plan to use Shopify Payments (eliminates the 0.5–2% extra transaction fee)
⚙️ Choose Custom Build when...
  • The unified customer record (shop + coaching + video) is core to the business model
  • Corporate branded orders and custom B2B workflows are a meaningful revenue stream
  • Video subscriptions with Mux access gating are a Phase 1 feature, not Phase 2
  • Long-term data ownership and no platform rent is a strategic priority
  • The coaching booking and accessory purchase experience must be seamlessly connected
  • You have technical oversight (CTO capacity) to hold the vendor accountable
⚠️ Shopify risks to accept
  • Coaching sessions require a bolt-on booking app that doesn't share a customer record with the shop
  • Video subscriptions require a separate platform (Teachable/Kajabi) at $119–$199/month
  • Corporate B2B workflow is not natively supported below Shopify Plus ($2,300/month)
  • Extra 0.5–2% transaction fee if not using Shopify Payments (Stripe preferred)
  • Platform lock-in: migrating customer and order data later is painful
🔴 Custom build risks to accept
  • Returns and fulfilment workflow not in current SOW — needs explicit scoping with Bihayy
  • Inventory race conditions on simultaneous last-unit purchases require careful implementation
  • Every operational improvement (discount stacking, bulk operations) is a dev ticket
  • 4–6 month build timeline delays first revenue compared to Shopify
  • Ongoing dependency on Bihayy retainer for anything beyond content management
Final recommendation
The architecture is already correct.
The scope needs two additions.

The custom Flutter + Supabase build is the right long-term architecture for GolfVault because GolfVault is not a standard ecommerce business — it is a three-business-line platform where the unified customer record is the product. Shopify cannot deliver that without an integration complexity that costs more than the custom build itself.


However, two order management gaps need to be added to the Bihayy SOW before signing:


1. Returns & refund workflow — customer-initiated return authorisation, Stripe refund trigger, and admin confirmation. This is not in the current $4,870 SOW and is not optional for an accessories business. Get a line-item quote from Bihayy before signing.


2. Shipping integration — EasyPost or ShipStation API for USPS/UPS label generation from the admin dashboard. Without this, Marcus is manually copy-pasting tracking numbers. Get a quote for this as a Phase 1 addition or accept it as a manual process at launch.


The catalog architecture — Payload CMS + Supabase + Algolia — is definitively better than Shopify for GolfVault's three-business-line catalog. There is no Shopify configuration that handles accessories, coaching sessions, and video courses in a unified search index with cross-sell logic. That decision is correct as specified.
